Project Title:
Simple Banking System using C
# Description
This project is a simple console-based banking system written in C.
It allows a user to:
Deposit money
Withdraw money
View updated balance after each operation
Perform up to 10 operations (Deposit/Withdraw)
The program uses basic C concepts like functions, loops, conditional statements, and user input validation.
# Features
✔ Deposit Money
User enters an amount to deposit
Amount must be greater than 0
Updated balance is displayed
✔ Withdraw Money
User enters amount to withdraw
Checks if balance is sufficient
Prevents negative or zero withdrawals
✔ Operation Limit
Maximum of 10 operations allowed
Invalid choices do not reduce the operation count
✔ Exit Anytime
User can exit the program before completing all operations
# How the Program Works
The user enters an initial starting balance
A loop runs (max 10 times) showing the menu:
Deposit
Withdraw
Exit
Based on the user’s choice, the respective function executes
Balance updates after every valid operation
Final balance is displayed before exiting
